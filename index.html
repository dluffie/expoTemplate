Project Title
AI-Assisted College Activity Point Management System
📖 Abstract
  • The AI-Assisted Activity Point Management System (APMS) is a full-stack web and mobile application designed to digitize and automate the college activity point workflow
  • Students upload certificates which are analyzed by Google Gemini AI for automatic data extraction
  • Teachers verify submissions and administrators manage rules, users, and analytics — all through role-based dashboards
🎯 Objectives
  • Build a centralized platform for managing student activity points across departments
  • Automate certificate data extraction using Google Gemini AI with OCR fallback
  • Enable teacher verification workflow — approve, reject, or request resubmission
  • Provide real-time dashboards with analytics for students, teachers, and admins
  • Implement configurable rules engine for activity point calculation with tiered modifiers
  • Support email notifications via Brevo/Resend/Nodemailer for reminders and status updates
  • Deliver a cross-platform mobile app using React Native Expo
  • Ensure data integrity with permanent activity snapshots and audit logging
👥 System Users & Roles
  • Students — Upload certificates for AI extraction, track earned activity points, view submission status, and manage profile
  • Teachers / Class Tutors — Verify and approve/reject submissions, submit activities on behalf of students, manage class subscriptions, send reminders
  • Administrator — Manage all users, configure rules engine, assign teachers to classes, view system-wide analytics, and audit logs
🛠️ Technology Stack
  • Frontend: React.js (Vite), CSS3
  • Mobile: React Native (Expo)
  • Backend: Node.js + Express.js 5
  • Database: MongoDB (Mongoose ODM)
  • AI Engine: Google Gemini API (@google/generative-ai)
  • File Storage: Cloudinary (image/document uploads)
  • Email: Brevo + Resend + Nodemailer
  • Auth: JWT + bcrypt + Email OTP Verification
  • Deployment: Render (backend + static frontend)
⚡ Key Features
  • Student registration with email OTP verification and password reset
  • Certificate upload with AI-powered data extraction (Gemini API)
  • Manual activity entry with rich form inputs
  • Teacher verification panel — approve, reject, or request resubmission with remarks
  • Faculty-initiated activity submission on behalf of students
  • Email reminders for pending submissions and approvals
  • Configurable rules engine with category-based point tiers
  • Role-based access control (Student / Teacher / Admin)
  • Admin analytics dashboard with charts and statistics
  • Audit logging for all critical system actions
  • Student profile management with Cloudinary image uploads
  • Cross-platform mobile application (Expo)

  • Reduces administrative workload while giving students a transparent way to track their academic progress
  • The cross-platform mobile app extends accessibility beyond the web
